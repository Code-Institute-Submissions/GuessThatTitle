WHO ARE WE?

Guess That Title is an online quiz game developed as a project to demontrate progress in learning HTML5, CSS3, and Javascript.

BEHIND THE SCENES

The GTT engine runs off Javascript. It draws all information from the OMDb API - a large database of films and tv shows. It also draws resources from JQuery and Bootstrap libraries.
